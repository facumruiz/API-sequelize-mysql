# API-sequelize-mysql
| Methods | Urls | 	Actions |
| :---:         |     :---:      |          :---: |
| GET   | /products     | List of products    |
| GET     | /products/{id}       | View a product      |
| POST    | /pruducts       | Create new product      |
| PUT     | /products/{id}       | Update a product      |
| DELETE     | /products/{id}       | Delete a product      |
